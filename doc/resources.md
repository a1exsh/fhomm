# Game Resources

## Icons

- `advbtns.icn`: adventure map: next hero, continue movement, kingdom overview, end turn, actions, save/load
- `advmice.icn`: adventure map: mouse pointer shapes
- `apanel.icn`: adventure actions: Map, Puzzle, Cast, Dig, and the OKAY button
- `art*.inc`: artifacts
- `barb32.icn`: the Hero Barbarian movements for the adventure map
- `btncmpgn.icn`: select new campaign, and the big CANCEL buton
- `btncom.icn`: network game COM port selection
- `btnhotst.icn`: select number of players for the Hot Seat (plus the CANCEL button)
- `btnmain.icn`: load game, new game, view high scores, view credits, quit
- `btnmodem.icn`: modem game
- `btnmp.icn`: multiplayer modes: Hot Seat, Network, Modem, Direct Connect (and its own CANCEL button)
- `btnnet.icn`: network game mode: Host or Guest (and the CANCEL)
- `btnnewgm.icn`: Standard, Campaign, Multiplayer, CANCEL

- `building.icn`: all common and town-specific buildings for the Build Structure window
- `buybuild.icn`: building blocks for rendering a pop-up dialog window of arbitrary size (with the Lions hoding a golden ball at the top)
- `campaign.icn`: campaign progress, OKAY, and the Restart Scenario button

- `castle0?.icn`: walls for the Castle Defense

- `clop32.icn`: fog of war for the adventure map
- `cmbtmous.icn`: battle: mouse pointer shapes
- `cpanel.icn`: control panel: New Game, Load, Save, Quit, OKAY, INFO, as well as controls for the Sound, Music, Movement speed, etc.
- `crst00??.icn`: Coat of Arms
- `dock.icn`: docks for the Town Screen
- `expmrl.icn`: Experience, Luck, and Morale icons
- `farm*.icn`: The Knight's Castle ("farm")
- `frst*.icn`: The Sorceress' Castle ("forest")
- `heroscrn.icn`: Hero Status Screen, including the vertical EXIT and DISMISS buttons
- `hiscore.icn`: High Scores Screen, including the vertical STANDARD/CAMPAIGN and EXIT buttons
- `hourglas.icn`: Hourglass for the waiting the enemy side's turn
- `keep0?.icn`: the tower and the projectile for the Castle Defense
- `kngt32.icn`: the Hero Knight movements for the adventure map
- `legend.icn`: a lone EXIT button
- `letters.icn`: W, M, O, S, C, J, G (for the resource indicators on the View World Screen)
- `locators.icn`: hired heroes and taken town buttons for the adventure map
- `losecmbt.icn`: a lone sad OKAY button
- `mobility.icn`: movement points indicators
- `moraleb.icn`: low ("bad") morale animation
- `moraleg.icn`: high ("good") morale animation
- `mtn6.icn`: mountains at lowest resolution (for the View World Screen?)
- `mtn32.icn`: mountains at high resolution for the adventure map
- `mtn*.icn`: the Warlock's Castle ("mountains")
- `netbox.icn`: TBD: single red bar (network game only?)
- `newgame.icn`: select map, difficulty, color, and, computer opponent level for the new game; also CANCEL and OKAY buttons
- `obj32-0?.icn`: adventure map objects
- `object32.icn`: TBD: castle moat and bridge, tavern(?), etc.
- `overview.icn`: Kingdom Overview Screen icons
- `overlay32.icn`: TBD: similar to object32
- `pln*.icn`: the Barbarian's Castle ("plains")
- `poof.icn`: Ghost disappearing(?)
- `rainbluk.icn`: good luck rainbow animation
- `recruit.icn`: more (arrow up), less (arrow down), MAX, CANCEL, and OKAY buttons
- `reddeath.icn`: just like poof, but red
- `request.icn`: OKAY and CANCEL buttons, also the purple bar
- `resource.icn`: kingdom resources
- `ressmall.icn`: a single icon with all resources and castles/towns
- `sceninfo.icn`: a lone OKAY button
- `scroll.icn`: scrollbar controls
- `scroll2.icn`: the same, but with black background
- `selector.icn`: TBD: red ellipse
- `smcrest.icn`: smaller (25x25) Coats of Arms
- `sorc32.icn`: the Hero Sorceress movements for the adventure map
- `spheres.icn`: TBD: blue, green, red, yellow, and white circles (11x11)
- `stoneback.icn`: background with a gray stone texture
- `strip.icn`: catle/hero army strips and backgrounds
- `sunmoon.icn`: sun and moon animation for the new day/week
- `surrendr.icn`: ACCEPT or DECLINE surrender
- `swapbtn.icn`: the EXIT button and TBD MONO icon
- `system.icn`: OKAY, CANCEL, YES, NO, OKAY, CANCEL once again, and the purple bar
- `tent.icn`: hero's tent in the battle
- `textbar.icn`: battle log bar and the AUTO, and SKIP buttons
- `vgenback.icn`: TBD: fancy Coats of Arms
- `viewarmy.icn`: view or dismiss troop
- `viewgen.icn`: hero's "General" options during a battle
- `wincmbt.icn`: animation after winning a battle
- `winloseb.icn`: border for the artifact capture after winning a battle
- `wrlk32.icn`: the Hero Warlock movements for the adventure map
